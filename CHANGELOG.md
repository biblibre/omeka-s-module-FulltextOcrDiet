# Changelog

## 1.0.0

- Renamed module to FulltextDiet
- Multiple properties to exclude from fulltext indexation can be selected via
  the administration interface using the native property selector
- Default exclusion: `extracttext:extracted_text`

## 0.0.1

- Initial release (SkipExtractText)
- Hardcoded exclusion of `extracttext:extracted_text` from fulltext indexation
